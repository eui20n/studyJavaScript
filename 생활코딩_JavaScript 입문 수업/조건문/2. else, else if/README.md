# else, else if
* else
  ```
    if(조건문){
      실행문;
    } else {
      // if가 true가 아니면 실행
      실행문;
    }
  ```
  
* else if
  ```
      if(조건문){
      실행문;
    } else if(조건문){
      // if가 아닐때
      실행문;
    } else if(조건문){
      // if가 아니고 위의 else if도 아닐때
      실행문;
    } else{
      // 위에 있는 모든 것이 아닐때
      실행문;
    }
  ```
