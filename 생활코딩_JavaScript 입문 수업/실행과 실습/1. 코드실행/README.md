<h1> 코드작성과 실행 </h1>
<p>기본적으로 웹브라우저에서 동작하는 자바 스크립트는 당연히 HTML안에 코드가 있어야함 => <script> 태그에 내용이 있음</p>
<p> 예시 </p>

```
<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8"/>
    </head>
    <body>
        <script>
            alert('Hello world');
        </script>
    </body>
</html>
```
